## Prerequisites

- React 16.8 or later
- Node.js 14 or later

## Installation

Install Motion via npm, yarn, or your preferred package manager:

```bash
npm install motion
```

Or with yarn:

```bash
yarn add motion
```

## Basic Setup

After installation, you can import Motion in your React components:

```jsx
import { animate } from "motion";
```

## Quick Example

Here's a simple example of using Motion in a React component:

```jsx
import { motion } from "motion/react";

export default function AnimatedBox() {
  return (
    <motion.div
      animate={{ rotate: 360 }}
      transition={{ duration: 1 }}
      style={{
        width: "100px",
        height: "100px",
        backgroundColor: "blue",
      }}
    />
  );
}
```
